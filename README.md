# MINTY

Personal financial management application leveraging the [Plaid API](https://plaid.com/) and the [MERN Stack](https://github.com/Hashnode/mern-starter).
![Minty-gif](minty.gif)

---
**Technologies**

Backend:
 - Express
 - Mongoose
 - MongoDB

Frontend:
 - React
 - Redux
 - Material-UI
 - Emotion
 - Animejs
